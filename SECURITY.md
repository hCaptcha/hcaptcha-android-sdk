# Security Policy

## Supported Versions

The last two major versions are officially supported.

Note: Enterprise customers of hCaptcha may receive support for older versions upon request.

| Version | Supported          |
| ------- | ------------------ |
| 5.x.x   | :white_check_mark: |
| 4.x.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Contact security@hcaptcha.com to report any security issues in the SDK.
